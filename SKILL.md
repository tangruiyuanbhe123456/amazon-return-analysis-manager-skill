# 亚马逊退货分析工具 (Amazon Return Analysis)

自动分析亚马逊卖家中心的退货数据，计算退货成本和趋势，生成品控改进方案。

## 功能

- 📊 **退货趋势分析**：分析退货率变化趋势，发现异常波动
- 💰 **退货成本计算**：自动计算FBA退货费、弃置费、损耗费
- 🔍 **退货原因分类**：将退货原因归为品质/尺寸/物流/描述不符等类别
- 🛠 **品控改进建议**：根据退货数据生成产品改进方案
- 📈 **竞品对比**：对比同类目的行业平均退货率

## 使用方法

### 1. 分析月度退货报告

把亚马逊后台下载的退货CSV数据粘贴给Agent：
```
分析这份退货报告：[粘贴CSV数据]
```

返回：
- 退货率 vs 行业平均
- 退货原因TOP5
- 估算FBA损失金额
- 紧急改进项

### 2. 单品退货深度分析

```
分析这个ASIN的退货情况：B0XXXXXX
```

返回：
- 该ASIN退货率趋势
- 主要退货原因
- 与同类目对比
- 改进优先级

### 3. 退货成本核算

```
计算这批退货的总成本：[粘贴数据]
```

返回：
- FBA退货处理费
- 弃置/清货费
- 库存损失
- 总损失估算

## 退货原因分类体系

| 类别 | 常见原因 | 应对策略 |
|------|---------|---------|
| 品质问题 | 损坏/瑕疵/不工作 | 加强质检，改进包装 |
| 尺寸问题 | 太大/太小/不合身 | 优化尺码表，加视频 |
| 描述不符 | 颜色不同/功能缺失 | 核对Listing描述与实物 |
| 物流问题 | 延迟送达/包装破损 | 换物流商，加强包装 |
| 不适用 | 买错/送错/改变主意 | 无法控制，但可以优化标题 |
